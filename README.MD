Swiggy Sales Data Analysis Dashboard (Excel)

Project Overview

This project analyzes Swiggy food delivery sales data using Microsoft Excel.
The objective of this project is to transform raw data into meaningful insights using data analysis techniques and interactive dashboard visualization.

The dashboard helps in understanding sales trends, customer ratings, order distribution, and category performance.

Tools & Technologies Used

- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Charts and Graphs
- Interactive Slicers
- Data Visualization

Key Insights

- Total sales and order analysis
- Monthly sales trends
- Daily sales trends
- Food category distribution (Veg vs Non-Veg)
- State-wise sales performance
- Top cities contributing to overall sales
- Average order value and rating analysis

Project Files

- swiggy_sales_dashboard.xlsx – Complete Excel project including dataset, analysis, and dashboard
- dashboard.png – Final dashboard visualization
- analysis1.png – analysis5.png – Data analysis screenshots
